AssemblyScript คือคอมไพเลอร์ที่แปลง **TypeScript Subset ที่เข้มงวด** ให้เป็น WebAssembly ถ้าคุณรู้จัก TypeScript อยู่แล้ว คุณสามารถเริ่มใช้ AssemblyScript ได้ในไม่กี่นาที ไม่มี Borrow Checker ของ Rust ไม่มีการจัดการ Memory แบบ C — เพียงแค่ TypeScript ที่มี Type และ Annotation เฉพาะของ Wasm

## ติดตั้ง AssemblyScript

```bash
# สร้างโปรเจกต์ Node.js ใหม่
mkdir as-demo && cd as-demo
npm init -y

# ติดตั้งคอมไพเลอร์ AssemblyScript
npm install --save-dev assemblyscript

# สร้างโครงสร้างโปรเจกต์ (สร้าง assembly/ และ tsconfig.json)
npx asinit .
```

ไดเรกทอรี `assembly/` เก็บไฟล์ Source `.ts` ของคุณ และ `asconfig.json` กำหนดค่าคอมไพเลอร์

## เขียนฟังก์ชัน AssemblyScript

เปิด `assembly/index.ts`:

```ts
// การบวกเลขจำนวนเต็ม — สังเกตประเภท i32 ที่ระบุชัดเจน
export function add(a: i32, b: i32): i32 {
  return a + b;
}

// Factorial — แบบ Recursive, ใช้ i64 เพื่อป้องกัน Overflow สำหรับ Input ขนาดใหญ่
export function factorial(n: i64): i64 {
  if (n <= 1) return 1;
  return n * factorial(n - 1);
}

// Clamp ค่า Float ระหว่าง lo และ hi
export function clamp(value: f64, lo: f64, hi: f64): f64 {
  if (value < lo) return lo;
  if (value > hi) return hi;
  return value;
}
```

ความแตกต่างหลักจาก TypeScript ทั่วไป:

- Type อย่าง `i32`, `i64`, `f32`, `f64` คือ **Numeric Type ของ Wasm** ไม่ใช่ `number` ของ TypeScript
- ไม่มี `any`, ไม่มี `unknown` และไม่มีการแปลง Type โดยอัตโนมัติ
- `export` ทำให้ฟังก์ชันเข้าถึงได้จาก JavaScript (เหมือน `(export ...)` ใน WAT)

## คอมไพล์ด้วย asc

```bash
# Development Build (อ่านได้ง่าย, ไม่ Optimise)
npx asc assembly/index.ts --outFile build/release.wasm --textFile build/release.wat

# Release Build (Optimise แล้ว, Binary เล็กกว่า)
npx asc assembly/index.ts --outFile build/release.wasm --optimize

# พร้อม Binding สำหรับใช้เป็น ES Module
npx asc assembly/index.ts --outFile build/release.wasm --bindings esm
```

## เรียกใช้จาก JavaScript

```js
// ใช้ @assemblyscript/loader Helper
import { instantiateStreaming } from '@assemblyscript/loader';

const { exports } = await instantiateStreaming(fetch('./build/release.wasm'));

console.log(exports.add(10, 32));            // 42
console.log(exports.factorial(BigInt(10)));  // 3628800n (i64 → BigInt)
console.log(exports.clamp(1.5, 0.0, 1.0));  // 1
```

หรือใช้ WebAssembly API โดยตรงโดยไม่ต้องใช้ Loader:

```js
const response = await fetch('./build/release.wasm');
const bytes = await response.arrayBuffer();
const { instance } = await WebAssembly.instantiate(bytes, {});

console.log(instance.exports.add(3, 4));  // 7
```
