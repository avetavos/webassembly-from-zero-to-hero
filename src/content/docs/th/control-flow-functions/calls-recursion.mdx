ฟังก์ชันใน WebAssembly เรียกกันด้วยคำสั่ง `call` argument จะถูก push ขึ้น stack ก่อนเรียก และค่าที่คืนกลับ (ถ้ามี) จะถูกฝากไว้บน stack เมื่อ callee return Wasm รองรับ direct recursion อย่างสมบูรณ์ — ฟังก์ชันสามารถเรียกตัวเองได้ — และ mutual recursion ระหว่างฟังก์ชันที่กำหนดใน module เดียวกัน

## คำสั่ง call

`call $funcname` เรียกฟังก์ชันที่มีชื่อนั้น สามารถเรียกด้วย index ได้ด้วย (`call 0`) แต่ named reference อ่านง่ายกว่ามาก ก่อนรัน `call` ให้ push argument ทั้งหมดตามลำดับจากซ้ายไปขวา:

```wasm
(module
  (func $double (param $n i32) (result i32)
    local.get $n
    i32.const 2
    i32.mul)
  (func (export "quadruple") (param $x i32) (result i32)
    local.get $x
    call $double    ;; passes $x; result replaces it on stack
    call $double))  ;; doubles again
```

`quadruple(3)` เรียก `$double` สองครั้ง ได้ผลลัพธ์ `3 → 6 → 12`

## Recursive functions

WAT อนุญาตให้ฟังก์ชันเรียกตัวเองได้ รูปแบบเหมือนทุกภาษา: push argument, call self, รวม result base case ใช้ `if` เพื่อ return ตรงๆ โดยไม่มี recursive call

```wasm
(module
  (func $factorial (export "factorial") (param $n i32) (result i32)
    local.get $n
    i32.const 1
    i32.le_s          ;; n <= 1?
    if (result i32)
      i32.const 1     ;; base case: 0! = 1! = 1
    else
      local.get $n
      local.get $n
      i32.const 1
      i32.sub
      call $factorial ;; factorial(n-1)
      i32.mul         ;; n * factorial(n-1)
    end))
```

ฟังก์ชันนี้มีทั้ง internal name (`$factorial`) เพื่อเรียกตัวเองได้ และ export name เพื่อให้ JavaScript เรียกได้

## ตัวอย่าง Fibonacci

การมี recursive call สองครั้งใน `else` arm ทำงานในลักษณะเดียวกัน — push argument, call, แล้วรวมผลลัพธ์ทั้งสอง:

```wasm
(module
  (func $fib (export "fib") (param $n i32) (result i32)
    local.get $n
    i32.const 1
    i32.le_s          ;; n <= 1?
    if (result i32)
      local.get $n    ;; fib(0)=0, fib(1)=1
    else
      local.get $n
      i32.const 1
      i32.sub
      call $fib       ;; fib(n-1)
      local.get $n
      i32.const 2
      i32.sub
      call $fib       ;; fib(n-2)
      i32.add
    end))
```

## ทดลองรัน

export const callsRecursionCode = `const wat = \`(module
  (func $factorial (export "factorial") (param $n i32) (result i32)
    local.get $n
    i32.const 1
    i32.le_s
    if (result i32)
      i32.const 1
    else
      local.get $n
      local.get $n
      i32.const 1
      i32.sub
      call $factorial
      i32.mul
    end))\`;
const bytes = await compileWat(wat);
const { instance } = await WebAssembly.instantiate(bytes, {});
console.log('factorial(5)  =', instance.exports.factorial(5),  '(expected 120)');
console.log('factorial(10) =', instance.exports.factorial(10), '(expected 3628800)');`;
