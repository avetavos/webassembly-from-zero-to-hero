`loop` ใน WebAssembly **ไม่วนซ้ำอัตโนมัติ** มันเป็นเพียง labeled construct ที่จุดเริ่มต้นเป็น branch target การวนซ้ำจะเกิดขึ้นเมื่อวาง `br_if` ไว้ที่ท้าย body เพื่อกระโดดกลับไปที่ loop label เมื่อ condition ยังเป็นจริง

## Loop construct

`(loop $label ...)` กำหนดจุดเริ่มต้นของ loop `br $label` ภายใน body จะกระโดดกลับไปที่จุดเริ่มต้น — ตรงข้ามกับทิศทางของ `br` ใน `block` การออกจาก loop ทำได้โดยให้ execution ไปถึง `end` (fall-through) หรือกระโดดไปที่ outer label

```wasm
(module
  (func (export "countDown") (param $n i32)
    (loop $again
      ;; body here
      local.get $n
      i32.const 1
      i32.sub
      local.tee $n     ;; decrement $n and keep value on stack
      i32.const 0
      i32.gt_s
      br_if $again)))  ;; repeat while $n > 0
```

## ตัวอย่าง counted loop: sumTo

ฟังก์ชันด้านล่างรวมจำนวนเต็มตั้งแต่ 1 ถึง `n` โดยใช้ `loop` local `$i` เป็น counter และ `$acc` สะสมผลรวม

```wasm
(module
  (func (export "sumTo") (param $n i32) (result i32)
    (local $i i32)
    (local $acc i32)
    i32.const 1
    local.set $i      ;; i = 1
    i32.const 0
    local.set $acc    ;; acc = 0
    (loop $again
      local.get $i
      local.get $acc
      i32.add
      local.set $acc  ;; acc += i
      local.get $i
      i32.const 1
      i32.add
      local.set $i    ;; i++
      local.get $i
      local.get $n
      i32.le_s
      br_if $again)   ;; repeat while i <= n
    local.get $acc))  ;; return acc
```

## ทดลองรัน

export const loopsCode = `const wat = \`(module
  (func (export "sumTo") (param $n i32) (result i32)
    (local $i i32)
    (local $acc i32)
    i32.const 1
    local.set $i
    i32.const 0
    local.set $acc
    (loop $again
      local.get $i
      local.get $acc
      i32.add
      local.set $acc
      local.get $i
      i32.const 1
      i32.add
      local.set $i
      local.get $i
      local.get $n
      i32.le_s
      br_if $again)
    local.get $acc))\`;
const bytes = await compileWat(wat);
const { instance } = await WebAssembly.instantiate(bytes, {});
console.log('sumTo(5)  =', instance.exports.sumTo(5),  '(expected 15)');
console.log('sumTo(10) =', instance.exports.sumTo(10), '(expected 55)');`;

<Callout title="loop ไม่วนซ้ำอัตโนมัติ">

</Callout>
